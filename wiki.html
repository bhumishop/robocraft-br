Introdução

Este wiki documenta como construir robôs com inteligência artificial usando componentes disponíveis no Brasil. O projeto usa DimOS como plataforma principal de IA, integrado com Modal GLM-5.1 para processamento de linguagem natural.

Nota: Este guia assume conhecimento básico de Arduino e programação Python. Se você é iniciante, recomendamos começar com o projeto ESP32-CAM.

DimOS Integration

DimOS é a plataforma de IA para robôs. Integra com Modal GLM-5.1 via API compatível com OpenAI.

Instalação

pip install uv
uv pip install dimos

# Configure no .env
MODAL_API_KEY=sua_chave_aqui

Uso Básico

from dimos.agents.modal_agent import agent

meu_agente = agent()
resposta = meu_agente.chat("Olá, qual é o seu nome?")
print(resposta)

Exemplo: Robô Falante

import edge_tts
import serial
import asyncio

arduino = serial.Serial('/dev/ttyUSB0', 9600)

async def speak(texto):
    # Gerar áudio
    tts = edge_tts.Communicate(texto, "pt-BR-FranciscaNeural")
    await tts.save("/tmp/fala.mp3")
    
    # Enviar para Arduino
    arduino.write(b"PLAY\n")

# Main loop
async def main():
    while True:
        input_text = input("Você: ")
        # Processar com DimOS
        response = agent.chat(input_text)
        await speak(response)

asyncio.run(main())

Troubleshooting

ESP32-CAM não conecta

  • Verifique GPIO0 conectado a GND durante upload
  • Use FTDI com 5V (não 3.3V)
  • selecione correto board no Arduino IDE
Problema: "Brownout detector"
Solução: Fonte USB não fornece corrente suficiente. Use fonte externa 5V 2A.

Servos não respondem

  • Verifique frequência PWM: 60Hz
  • Ângulo entre 150-600 (não 0-180)
  • Alimentação externa se +4 servos

DFPlayer não reproduz

  • Arquivos devem ser: 001.mp3, 002.mp3...
  • Cartão formatado em FAT32
  • Use resistor 1k no RX

Wi-Fi não conecta

  • Rede 2.4GHz (não 5GHz)
  • Verifique SSID e senha
  • Signal strength mínimo -70dBm

FAQ

Preciso de internet?

Não! Use TTS local (Piper, Coqui, eSpeak NG) para robôs offline.

Melhor TTS para robôs autônomos?

Piper TTS oferece melhor combinação qualidade + latência (<50ms).

Posso usar outros modelos de IA?

Sim! DimOS é compatível com qualquer API OpenAI-like (GPT-4, Claude, etc).

Qual Arduino escolher?

  • Uno: Projetos básicos
  • Mega: Muitos servos/sensores
  • ESP32: Wi-Fi + Bluetooth
